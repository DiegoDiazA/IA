Instrucciones:
Este proyecto requiere tener instaladas las siguientes librerías para funcionar:
-Flask
-Pillow
-Tensorflow
-Numpy

Una vez que ya se tienen las librerías, ya es cuestión de ejecutar el archivo de app.py 
desde visual studio, donde al ejecutarse mostrará en la terminal el enlace al servidor
local donde se encuentra el proyecto, y ya dentro de la pagina web solo hace falta
cargar las imagenes para que la ia decida con el uso de algoritmos CNN qué tipo de roca 
es la de la imagen.
